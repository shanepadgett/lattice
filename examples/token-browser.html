Line heights

tight · 1.1

Tokens create a shared language between design and code. With tight line height, lines compress and the paragraph becomes denser, revealing how space affects the rhythm of reading across UI copy.

snug · 1.25

Tokens create a shared language between design and code. Snug line height keeps paragraphs compact while leaving just enough air to preserve readability in longer passages and instructions.

normal · 1.5

Tokens create a shared language between design and code. Normal line height is balanced and versatile, suitable for product copy, documentation, and comfortable reading at most sizes.

relaxed · 1.625

Tokens create a shared language between design and code. Relaxed line height opens the paragraph, increasing clarity and making long-form content feel spacious and editorial.

loose · 2

Tokens create a shared language between design and code. Loose line height gives the most air, emphasizing calm, approachable reading when content needs a gentle pace.
